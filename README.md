## The Context Field тАФ Where the Magic Happens

The third input тАФ **The Personalisation** тАФ is the most powerful and most 
underused part of MagiTeacher.

The Context field is a free-text space where you tell the AI exactly what 
angle, theme, or depth you want. The same topic can produce radically 
different resources depending on what you write here.

- ЁЯОи **Add a theme** тАФ "Frame this around a Spider-Man mission scenario"

- ЁЯОп **Add exam precision** тАФ "Focus on AO1 and AO3, use mark scheme language"
